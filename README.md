# pytorch_fea

The repo is the refactored code of our paper published at Computer Methods and Programs in Biomedicine, titled "PyTorch-FEA: Autograd-enabled Finite Element Analysis Methods with Applications for Biomechanical Analysis of Human Aorta"

The orignal code of the paper is available at 
https://github.com/liangbright/pytorch_fea_paper

The prepint of our paper is https://www.biorxiv.org/content/10.1101/2023.03.27.533816v1
